Knight and Sword Asset Documentation

This package contains two simple scenes for preview and setup. Follow the steps below to use this asset:

1. Import the package into your Unity project.
2. Open the "Demo Scene" to see the knight and sword in action.
   - This scene shows a simple display example.
3. Open the "Sample Scene" to test rig and weapon setup.
   - Drag the Sword prefab into the character's right hand.
   - Adjust the sword's position and rotation if necessary.
4. Press Play to preview the animation and weapon placement.
5. Optional: You can duplicate the Sword prefab to use multiple swords in the scene.

Notes:
- All Prefabs have their root Transform set to position (0,0,0), rotation (0,0,0), and scale (1,1,1) for Asset Store compliance.
- No additional setup or coding is required to use the knight and sword in your scenes.
